# RTRP Project – Early Disease Detection

## 📌 Project Overview
This project predicts the risk of disease using machine learning based on medical parameters like cholesterol, blood pressure, etc.

## 🎯 Objective
To detect diseases at an early stage using data instead of symptoms.

## 🧠 Algorithms Used
- KNN
- SVM
- Random Forest (Final Model)
- Gradient Boosting
- ANN

## ⚙️ Technologies Used
- Python
- Machine Learning
- Pandas, NumPy, Scikit-learn
